### 개발 환경
* NodeJS (v12.9.1)
* Visual Studio Code

### Available Scripts
`.env` 파일을 생성하여 다음과 같은 내용을 입력합니다.  
```
TOKEN = 발급받은 봇 TOKEN
```
봇 `Token`은 [여기](https://discordapp.com/developers/applications/&prev=search)에서 발급 받을 수 있습니다.

`npm install`  
봇 실행에 필요한 패키지들을 설치합니다.  
  
`npm start`
봇을 실행합니다.
